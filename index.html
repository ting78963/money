📊 出場訊號條件
🟢 緊急全出
外資爆量賣超（≥3倍基準）
或股價跌破止損-10%
🟢 建議全出
外資連賣3天以上 + 評分≥3
或出貨型上漲（外資大賣+股價漲+量放大）
🟠 建議減碼
外資明顯賣超（≥1.5倍基準）
或持倉15天以上 + 破10日線
🟡 留意觀察
外資正常賣超（≥0.5倍基準）
或破5日線+量放大
或持倉超過20天
🔴 建議續抱
外資連續2天有意義買超（標準化分數>0.3）
🔴 加碼機會
外資+投信同時連續3天有意義買超
⚪ 中性觀察
買賣超幅度不顯著，觀察為主
🟡 投信布局，留意觀察
投信連續3天買超但外資未確認，需謹慎
📐 均線判斷邏輯
持倉1-5天 + 破10日線 🟡 留意觀察
持倉6-15天 + 破10日線 🟡 留意觀察
持倉15天+ + 破10日線 🟠 建議減碼
外資大賣 + 破10日線 🟢 建議全出
破5日線 + 量放大 🟡 留意觀察
⏱️ 持倉天數警示
持倉超過20天 🟡 留意觀察
持倉超過25天 🟠 建議減碼
波段策略目標持倉5-20個交易日
超過20天表示波段可能已結束，建議評估出場
⚠️ 出貨型上漲
條件：外資大賣（≥1.5倍基準）+ 股價上漲 + 量放大（≥1.5倍）

意義：表面上股價在漲，但主力正在利用散戶追高出貨。這是最危險的訊號，應考慮出場。
📊 族群訊號說明
族群買賣超採用標準化分數，讓台積電與群創等不同市值的股票能公平比較。

計算方式：
每支股票 = 今日買賣超金額 ÷ 過去10日平均買賣超金額
族群分數 = 族群內所有股票的平均標準化分數

訊號門檻：
分數 > 0.3 才算有意義的買超（過濾雜訊）
分數 < -0.3 才算有意義的賣超
📏 個股外資基準說明
個股持倉的「基準」是過去10日外資買賣超金額的平均絕對值。

例如：過去10日外資平均每天買賣 500萬元
今天賣超 750萬元 = 1.5倍基準 → 明顯賣超
今天賣超 1,500萬元 = 3倍基準 → 爆量賣超（緊急）
